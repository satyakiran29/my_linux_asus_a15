# Resource Monitoring with btop on ASUS TUF A15 (Ubuntu 26.04 LTS)

`btop` is a modern, interactive terminal-based resource monitor that displays usage and stats for processor, memory, disks, network, and processes. It features a highly responsive interface with full mouse support.

## Installation and Usage

Follow these steps to install and run `btop` on your system:

1. Update your system's package database:
   ```bash
   sudo apt update
   ```

2. Install `btop`:
   ```bash
   sudo apt install btop
   ```

3. Run the tool:
   ```bash
   btop
   ```

## Key Features

- **Interactive UI**: Supports mouse navigation to filter, sort, and terminate processes.
- **Detailed Graphs**: Displays real-time graphs for CPU core usage, memory allocation, disk read/write, and network traffic.
- **Customizable**: Built-in settings menu to change color themes, adjust layout panels, and toggle display options.
