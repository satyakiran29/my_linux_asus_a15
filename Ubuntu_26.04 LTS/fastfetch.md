# Installing and Running Fastfetch on ASUS TUF A15 (Ubuntu 26.04 LTS)

`fastfetch` is a modern, high-performance system information tool (a faster alternative to the older, unmaintained `neofetch`) that displays system hardware and software details alongside a clean ASCII logo of your Linux distribution.

## Installation and Usage

Follow these steps to update your package database, install `fastfetch`, and view your system specifications:

1. Update your system's package database:
   ```bash
   sudo apt update
   ```

2. Install `fastfetch`:
   ```bash
   sudo apt install fastfetch
   ```
   *(Note: You can install `neofetch` instead using `sudo apt install neofetch` if you prefer the classic tool, though it is deprecated and no longer actively maintained.)*

3. Run the tool to display system information:
   ```bash
   fastfetch
   ```
